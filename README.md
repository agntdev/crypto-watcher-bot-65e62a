# Crypto Watcher Bot

Telegram bot for private crypto watchlists: price & percent alerts, on‑demand prices, and optional daily summaries with quiet hours and per‑alert cooldowns.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
